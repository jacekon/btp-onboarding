**Extend SAP cloud solutions via Kyma runtime**

**SAP BTP, Kyma runtime** can be used to extend a variety of SAP cloud applications. The complete list is provided [here](https://help.sap.com/viewer/65de2977205c403bbc107264b8eccf4b/LATEST/en-US/346864df64f24011b49abee07bbd79af.html "CTRL + click to follow the link").   
  
The process of connecting an SAP cloud application to Kyma runtime differs depending on the solution you want to extend.  
The above link to SAP Help Portal will guide you through these different processes. 

**Create and Assign a Formation for SAP CX Solutions**

A Formation is used to assign one or more registered SAP CX solutions to the subaccount which contains Kyma runtime.  
Follow the steps that describe how to [assign SAP Systems to a Formation](https://help.sap.com/viewer/65de2977205c403bbc107264b8eccf4b/LATEST/en-US/68b04fa73aa740cb96ed380a85a4761a.html "CTRL + click to follow the link") to complete this process.
